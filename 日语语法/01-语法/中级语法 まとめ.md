# 001 ～とき・～直後に
  1. [[N2 001-1～際（に）]] 
     - [[動詞辞書形]]+ 際（に）  
	    　例：行く際に
	- [[動詞た形]] + 際（に）  
	    　例：帰った際に（更正式）
	- [[名詞の]] + 際（に）  
	    　例：出張の際に
     **正式版「～とき」 + 关键时机 + 书面表达**
	     👉强调：
			**带有“特定场合 / 关键时机 / 正式场面”的感觉**
		👉典型例句（建议背）：
			ご利用の際は、事前に予約してください。  
		    （使用时请提前预约）
  2. [[N2 001-2～に際して・～に当たって]] 
     - [[動詞辞書形]]+ に際して  
	 - [[名詞]] + に際して
		👉 に際して  
		= **重要时刻（人生节点）**
		👉 に当たって  
		= **做某事前（准备阶段）**
		👉强调：
			👉 に際して：  
			**“在这个重要人生节点”**
			👉 に当たって：  
			**“在做这件事之前（要准备）”**
		👉典型例句（建议背）：
			結婚に際して、家を買った。✔（人生大事）
			結婚に当たって、費用を準備した。✔（准备）
  3. [[N2 001-3～たとたん（に）]]
      [[動詞た形]]+ とたん（に）
		👉 ～たとたん（に）＝  
		**刚一……就……（瞬间 + 突发 + 非人为）**
		👉强调：
			 **“刚一……就……”（瞬间立刻发生）**
		👉典型例句（建议背）：
			- 立ち上がったとたん、めまいがした。  
			    （刚站起来就头晕了）
			- 窓を開けたとたん、強い風が入ってきた。
  4. [[N2 001-4～（か）と思うと・～（か）と思ったら]]
     [[動詞た形]] + かと思うと
     [[動詞た形]] + かと思ったら
		👉 ～（か）と思うと／と思ったら ＝  
		**刚一……就……（变化快 + 出乎意料 + 状态切换）**
		👉强调：
			 ① 瞬间变化（重点）
			 ② 动作连续发生（节奏快）
			 ③ 情绪 / 状态反复变化（高频）
			 变化很快，有“看起来是A，结果马上变B”的感觉
			 ✔ 比「たとたん」多了：  
			 👉 **“认知/观察到变化”**
		👉典型例句（建议背）：
			子供は泣いたかと思ったら、すぐ笑い出した。
			電車は止まったかと思うと、また動き出した。
  5. [[N2 001-5～か～ないかのうちに]]
    [[動詞辞書形]]+ 动词ない形 + かのうちに
	[[動詞た形]]+ 动词ない形 + かのうちに
	    **刚要……还没来得及……就……（极限瞬间）**
	    👉强调：
			1. 时间几乎为0（极短）
			2. 前项还没完全发生
			3. 后项已经发生
		👉典型例句（建议背）：
			- ドアを開けるか開けないかのうちに、猫が飛び出した。
			- 彼は名前を聞くか聞かないかのうちに、もう忘れてしまった。

# 002  ~している

即将发生    ～ところだ
即将变化   ～ようとしている
刚刚结束    ～たところだ
正在进行    ～最中だ
正在变化   ～つつある  
| 正在变化 | つつある     |

一直变化   ～ばかりだ・一方だ
  1. [[N2 002-1～最中（さいちゅう）だ]]
     - [[動詞ている形]] + 最中だ  
    　例：勉強している最中だ
	- [[名詞の]]+ 最中だ  
    　例：会議の最中だ
        👉 ～最中だ   
			**正在……当中（而且是关键进行中）**
		👉 **“正在……当中 / 正处于最关键的过程中”**
		👉强调：
		- 动作正在进行
		- 而且是“关键时刻 / 正忙着
		👉典型例句（建议背）：
		- 授業の最中に、地震が起きた。
		- 今は会議の最中ですので、後でご連絡します。
  2. [[N2 002-2～うちに]]
	- [[動詞辞書形]] + うちに  
	    　例：忘れるうちに
	- [[動詞ない形]] + うちに  
	    　例：忘れないうちに
	- [[動詞ている形]] + うちに  
	    　例：見ているうちに
	- [[い形容詞]] + うちに  
	    　例：若いうちに
	- [[な形容詞]]+ な + うちに  
	    　例：元気なうちに
	- [[名詞の]]+ うちに  
	    　例：学生のうちに
     👉 ～うちに ＝  
		**在……期间（趁还没变 / 不知不觉变化）**
		👉强调：
		👉 「うちに」的本质：
		👉 **“在某个时间范围内（变化发生前 or 变化过程中）”**
		👉典型例句（建议背）：
		- 忘れないうちに、連絡してください。
		- 日本に住んでいるうちに、日本文化が好きになった。
  3. [[N2 002-3～ばかりだ・一方だ]]
     - [[動詞辞書形]] + ばかりだ
       👉 **“只是在不断…… / 一直朝某个方向发展”**
		强调：
		- 单方向变化
		- 没有停止的趋势
     - [[動詞辞書形]] + 一方だ
       👉 **“一直朝一个方向发展（而且多为不好的方向）”**
		👉强调：
		- 单方向变化
		- 通常带负面评价
     👉 ～ばかりだ・一方だ ＝  
		**一直朝一个方向发展（停不下来）**
		但：
		- ばかりだ → 主观 / 可好可坏
		- 一方だ → 客观 / 多为变坏
		| 项目        | ばかりだ   | 一方だ            |
		| 变化方向 | 单方向       | 单方向           |
		| 情感色彩 | 中性/主观  | 偏负面/客观   |
		| 好坏        | ✔ 都可以 | ❌ 多为不好  |
		| 语气        | 感叹/无奈  | 冷静描述       |
		👉典型例句（建议背）
		- 仕事は忙しくなるばかりだ。
		- 環境問題は深刻になる一方だ。
  4. [[N2 002-4～(よ)うとしている]]
     - [[動詞意向形（よう形）]]+ としている
       👉 **“正要…… / 即将……”**
		👉强调：
		- 动作就在眼前
		- 还没开始
		👉典型例句（建议背）：  
		- 出かけようとしているところだ。
	    （正要出门）
		- 寝ようとしているときに、電話が来た。  
	    （正要睡觉时电话来了）
     👉 ～（よ）うとしている 
		**正要…… / 试图……（临界点）**
		
  5. [[N2 002-5～つつある]]
     - [[動詞ます形（去ます）]]+ つつある
      👉 ～つつある ＝  
		**正在逐渐……（变化进行中）**
	    👉强调：
		- 变化正在进行
		- 但还没完成
		- 是一个过程
		 👉典型例句（建议背）：
		- 環境問題は深刻になりつつある。
		-  IT技術は急速に発展しつつある。
		    
  6. [[N2 002-6～つつ]]
     - [[動詞ます形（去ます）]]+ つつある
      👉 ～つつ ＝  
		**一边……一边…… / 虽然……但是……（书面）**
		- 用法① → 同时进行
		- 用法② → 内心矛盾
		👉强调：
		- 书面语（比「ながら」正式）
		- 主语必须一致（重要！）
		👉典型例句（建议背）：
		- 彼はテレビを見つつ、食事している。
		- 危険だと知りつつ、その道を選んだ

# 005 だけ
  1. [[N2 005-1～に限り]]
     名詞＋に限り
      👉 ～に限り ＝  
		**只限…… / 唯独……（范围限定）**）**
		👉强调：
			客观
		👉典型例句（建议背）：
			- 本日に限り、特別割引を行います。
			- 彼に限り、その話は信じられない。
  2. [[N2 005-2～限り（は）]]
	- [[動詞辞書形]] + 限り（は）
	- [[動詞ている形]] + 限り（は）
	- [[名詞の]] + 限り（は）
	- [[な形容詞な]] + 限り（は）
	- [[い形容詞]] + 限り（は）
      👉 ～限り（は）＝  
		**只要……就……（条件限定）**
		👉强调：
			在这个条件范围内，一切成立
		👉典型例句（建议背）：
		- 努力する限りは、成功できる。
		- 元気な限りは、働き続けたい。
  3. [[N2 005-3～限りでは]]
    - [[動詞普通形]]+ 限りでは
	- [[動詞ている形]] + 限りでは（最常见）
      👉 ～限りでは ＝  
		**根据目前掌握的信息来看（不完全判断）**
		👉强调：
			根据……来看 / 就……范围而言   不完全判断
		👉典型例句（建议背）：
			- 見た限りでは、この製品に問題はなさそうだ。
			- 聞いている限りでは、その計画は順調だ。
  4. [[N2 005-4～に限って]]
     [[名詞]] + に限って
      👉 ～に限って ＝  
		**偏偏…… / 唯独……（主观例外）**
		👉强调：
			==不满 / 抱怨 / 意外==
		👉典型例句（建议背）：
		- 大事な日に限って、体調が悪くなる。
		- 彼に限って、嘘をつくはずがない。

		| 项目  | に限って    | に限り    |
		| 含义  | 偏偏 / 唯独 | 只限     |
		| 语气  | ==主观==  | ==客观== |
		| 场景  | 抱怨/情绪   | 通知/规则  |

			- 彼の意見には説得力のあるものがある。
  5. [[N2 026-5～ことだ]]
     动词辞书形 + ことだ
     动词ない形 + ことだ
           # ⭐“最好……”
		   # ⭐“应该……”
		👉 ～ことだ ＝
		# “如果想达到目的，最好……”
		（经验性建议）
		👉强调：
			# ⭐“最好……”
			# ⭐“应该……”
			👉 ✔ 本质：
			# ⭐基于经验给出的建议/忠告
		👉典型例句（建议背）
			- 疲れているなら、早く寝ることだ。
			- 日本語が上手になりたければ、毎日勉強することだ。
			- 健康のためには無理をしないことだ。
			- 困ったら先生に相談することだ。
			- 合格したければ、復習することだ。
			
  6. [[N2 026-6～ことだろう・～ことか]]
     普通形 + ことだろう
        | 用法  | 含义     |
	    | 推量  | 一定……吧  |
	    | 感叹  | 该多么……啊 |
		|语法|含义|
		|～ことだろう|想必……吧／该多么|
		|～ことか|多么……啊|
		👉 核心：
		# “强烈感叹 + 感情推量”
		👉强调：
			# ⭐“多么……”
			# ⭐“该有多……”
			👉 ✔ 纯感叹表达
		👉典型例句（建议背）
		- 母は今ごろ心配していることだろう。
		- 合格してどんなに嬉しかったことだろう。
		- どれほど苦しかったことか。
		- 一人でどんなに寂しかったことか。
		- 彼も驚いたことだろう。
