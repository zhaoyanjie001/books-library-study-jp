一类形容词+ば
	把词尾的い变成ければ、いい→よければ
～と
	相当于汉语的 一。。。就。。。
	1. 当前句尾假设条件时，就会产生后句
		名詞だ／動詞普通形／い形容詞／な形容詞＋と
	2. と也可以表示恒常性状态，真理，反复性的状态，习惯等
～なら
	表示根据对方的言谈或者交谈的下场情况提出自己的意见或者观点
～たら
	表示假定条件相当于汉语的  要是。。。的话
	其接续方式时过去形式的た变成たら。但当前项接二类形容词和名词的过去否定式时要用~でなかったら
　
　区别：
	👉 **～たら：发生之后（时间顺序）**  
	👉 **～なら：假设 / 根据对方情况提出建议**
たら = 做了A之后 / 如果发生
なら = 如果是那样的话 / 那我建议
# 核心对比（最重要部分）

|项目|～たら|～なら|
|---|---|---|
|时间关系|✔ 有先后顺序|❌ 没有时间顺序|
|条件性质|现实条件|假设 / 话题|
|建议|❌ 较少|✔ 很常用|
|意外发现|✔ 可以|❌ 不行|
|根据对方话|❌ 不强调|✔ 很强|

# 最关键对比（考试重点）

## ① 命令 / 请求谁能用？

|语法|是否可以|
|---|---|
|～と|❌|
|～ば|❌|
|～たら|✔|
|～なら|✔（更自然）|
## ② 时间先后

| 语法  | 是否有“先A后B” |
| --- | --------- |
| ～と  | ✔（自动发生）   |
| ～ば  | ❌         |
| ～たら | ✔（明确）     |
| ～なら | ❌         |
## ③ “建议”谁最强？

👉  
🥇 ～なら（最自然）  
🥈 ～たら  
❌ ～ば / ～と（基本不用）

## ④ “发现”只能用谁？

👉  
✔ ～たら  
✔ ～と（少量情况）
❌ ～ば / ～なら

# 一句话总记忆（最重要🔥）

👉  
**と＝自然就发生（规律）**  
👉  
**ば＝条件成立（逻辑）**  
👉  
**たら＝做完再做（万能）**  
👉  
**なら＝如果是那样（建议）**

# 考试技巧（N2很关键）

👉 不会选的时候：

1️⃣ 有“命令 / 请求”  
👉 选：たら / なら

2️⃣ 有“发现”  
👉 选：たら

3️⃣ 有“建议 / 对话”  
👉 选：なら

4️⃣ 客观规律  
👉 选：と
