三者嗾使指示代词，该句由[[N5-N3 1.判断句／存在句#^54cad7]](~は～です)变化得到，意思是这个/那个
1. これ・それ・あれ
	1. 当说话人和听话人面对面，且中间间隔一段距离时
		これ：事物离说话人较近
		それ：事物离听话人较近
		あれ：事物离说话人，听话人都比较远
		
	2. 当说话人和听话人位于同一位置时
		これ：事物离说话听话人都较近
		それ：事物离说话听话人都较远
		あれ：事物离说话人，听话人都更远
		
2. その・その・あのは～です
	三者都是修饰语，后接名词。表示位置关系同上
	
3. ここ・そこ・あそこは～です
	 三者都表示场所，其表示的位置关系同上，它们的礼貌表达分别时
	 こちら、そちら、あちら
	 
4. だれ・なん・どこ・どれ・どの～ですか
	 誰ですか　用于询问人
	 何ですか　用于询问物
	 どこですか　用于询问地点
	 どれですか　用于三个以上的事物选择一个
	 どの～ですか　用于三个以上的事物选择一个

6. 指示代词

|      | 近称         | 中称         | 远称         | 不定称        |
| ---- | ---------- | ---------- | ---------- | ---------- |
| 指示事物 | これ         | それ         | あれ         | どれ         |
| 指示场所 | ここ         | そこ         | あそこ        | どこ         |
| 指示方向 | こちら<br>こっち | そちら<br>そっら | あちら<br>あっち | どちら<br>どっち |
| 连体词  | この<br>こんな  | その<br>そんな  | あの<br>あんな  | どの<br>どんな  |
| 副词   | こう         | そう         | ああ         | どう         |

これ、それ、あれ、どれ可单独使用，不可接名词
例：これ、それ、あれ、どれ＋は～ですか
その、その、あの、どの　和　こんな、そんな、あんな、どんな 不可单独使用，只可接在名词前
