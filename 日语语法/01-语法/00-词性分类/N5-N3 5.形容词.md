1. 一类形容词
以い结尾的形容词

|     | 现在肯定 | 现在否定   | 过去肯定   | 过去否定     |
| --- | ---- | ------ | ------ | -------- |
| 敬体  | 高いです | 高くないです | 高かったです | 高くなかったです |
| 简体  | 高い   | 高くない   | 高かった   | 高くなかった   |
2. 二类形容词
不以い结尾的形容词

|     | 现在肯定 | 现在否定      | 过去肯定  | 过去否定         |
| --- | ---- | --------- | ----- | ------------ |
| 敬体  | 便利です | 便利ではありません | 便利でした | 便利ではありませんでした |
| 简体  | 便利だ  | 便利ではない    | 便利だった | 便利ではなかった     |
3. 形容词修饰名词
一类形容词+名词
二类形容词+な＋名词

4. 名词1+は名词2が一类/二类形容词です
5. ～より／～のほうが／～ほど
	AはBより一类/二类形容词です
	表示A比B更。。。
	AよりBのほうが一类/二类形容词です
	表示B比A更。。。
	AはBほど一类形容詞くないです／二类形容詞ではありません
	表示A没有B。。。
