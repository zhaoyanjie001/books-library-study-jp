# 一、～ばかりか

## 1️⃣ 接续（必考）

- [[名詞]] + ばかりか
- [[動詞普通形]] + ばかりか
- [[い形容詞]] + ばかりか
- [[な形容詞]] + である + ばかりか

✔ 例：

- 雨ばかりか
- 遅れるばかりか
- 高いばかりか
- 不便であるばかりか

👉 ❗接续和「のみならず」很像（常一起考）

---

# 二、核心意思

👉 **“不仅……反而连……也……”**

👉 ✔ 本质：  
👉 **递进 + 出乎意料（负面居多）**

---

# 三、使用场景（重点）

---

## ✔ ① 递进（B更严重/更夸张）

- 雨ばかりか、風も強くなった。  
    （不仅下雨，风还更大）

---

## ✔ ② 负面结果叠加（最常见）

- 遅刻したばかりか、宿題も忘れた。  
    （不仅迟到，还忘了作业）

---

## ✔ ③ 强调“更糟糕的是”

👉 B常常比A更严重！

---

# 四、语感本质（非常重要）

👉 「ばかりか」＝

👉 **“不只是A，甚至连更糟的B也……”**

---

# 五、和相似语法区别（重点）

---

## ① ～ばかりか vs ～のみならず

|项目|ばかりか|のみならず|
|---|---|---|
|含义|不仅…甚至更…|不仅…而且…|
|情感|多负面|中性|
|强度|更强|较弱|

✔ 对比：

- 遅刻したばかりか、怒られた（更糟）
- 遅刻したのみならず、連絡もしなかった（并列）

---

## ② ～ばかりか vs ～だけでなく

|项目|ばかりか|だけでなく|
|---|---|---|
|含义|更强递进|普通并列|
|语气|强|中性|

---

## ③ ～ばかりか vs ～ばかりでなく

👉 很接近：

- ばかりでなく → 中性
- ばかりか → 更强调/更戏剧性

---

## ④ ～ばかりか vs ～に限らず

👉 完全不同：

- ばかりか → 递进
- に限らず → 范围扩展

---

# 六、核心限制（考试重点）

## ❗① 后项必须“更进一步”（必考）

👉 B比A更严重/更夸张

---

## ❗② 多用于负面（高频）

👉 常见：

- 遅刻
- ミス
- 問題

---

## ❗③ 书面表达

👉 不太口语

---

## ❗④ 接续范围广

👉 和「のみならず」一样：

- 名词 ✔
- 动词 ✔
- 形容词 ✔

---

# 七、考试高频考点

## ① 和「のみならず」混考（必考）

👉 技巧：

- 中性递进 → のみならず
- 负面递进 → ばかりか

---

## ② “更严重结果”识别

👉 关键词：

- その上
- さらに

👉 → 高概率「ばかりか」

---

## ③ 接续判断（重要）

👉 动词/形容词也能接 → 排除其他选项

---

## ④ 情感判断（关键）

👉 是否带抱怨/负面？

---

# 八、典型例句（建议背）

- 彼は遅刻したばかりか、連絡もしなかった。
- この店は高いばかりか、味もよくない。

---

# 九、一句话总结

👉 ～ばかりか ＝  
**不仅……反而连更糟的……也……**

---

# 十、帮你建立体系（非常重要）

👉 递进类语法对比：

| 语法    | 含义       |     |     |        |
| ----- | -------- | --- | --- | ------ |
| だけでなく | 普通并列     |     |     |        |
| のみならず | 正式递进     |     |     |        |
| ばかりか  | 强递进（多负面） |     | に限り | 只限（规则） |
| に限って  | 偏偏（主观）   |     |     |        |
| に限らず  | 不仅限      |     |     |        |
