# 2 言葉

## 訴える（うったえる）

### 含义：诉说、控诉、呼吁

1. 患者の家族は医療ミスで病院を訴えた。  
    → 患者家属因医疗事故起诉了医院。

【会社、学校、相手を〜】  
→ 起诉公司、学校、对方

2. A選手は試合後、腰の痛みを訴えた。  
    → A选手赛后诉说腰痛。

【症状、不安、不満を〜】  
→ 诉说症状、不安、不满

3. 市長は演説で市民に平和の大切さを訴えた。  
    → 市长在演讲中向市民呼吁和平的重要性。

【重要性、必要性を〜】  
→ 呼吁重要性、必要性

---

## 押さえる（おさえる）

### 含义：按住、控制、抑制、掌握

1. ドアが閉まらないように押さえておいてください。  
    → 请按住房门，不要让它关上。

【帽子、髪を〜】  
→ 按住帽子、头发

2. 女の人が苦しそうな顔でおなかを押さえている。  
    → 那位女性捂着肚子，表情很痛苦。

【口、耳、胸、傷口を〜】  
→ 捂住嘴、耳朵、胸口、伤口

3. 面接の前に押さえておきたい五つのポイント。  
    → 面试前想事先掌握的五个重点。

【要点を〜】  
→ 掌握要点

4. ダイエット用にカロリーを抑えた食品。  
    → 为减肥而制作的低热量食品。

【甘さ、費用、上昇、増加を〜】  
→ 抑制甜度、费用、上涨、增加

5. 怒りたい気持ちを必死で抑えた。  
    → 拼命压抑想发火的情绪。

【感情、怒りを〜】  
→ 抑制感情、怒气

---

## 傾く（かたむく）

### 含义：倾斜；偏向；衰退

1. 乗客が一か所に集まると、船が傾いてしまう。  
    → 乘客集中到一处的话，船会倾斜。

【建物、床、柱、体が〜】  
→ 建筑、地板、柱子、身体倾斜

2. 五時を過ぎて太陽が西に傾き始めた。  
    → 过了五点，太阳开始西斜。

【月、日が〜】  
→ 月亮、太阳西斜

3. 婚約者がいるのに、ほかの人に心が傾いている。  
    → 明明有未婚妻（夫），却对别人动了心。

【気持ち、考えが〜】  
→ 心情、想法偏向……

4. 不景気で売り上げが落ち、会社が傾き始めた。  
    → 因经济不景气销售额下降，公司开始衰败。

【商売、経営が〜】  
→ 生意、经营衰退

---

## 刻む（きざむ）

### 含义：切碎；刻上；铭记

1. ネギを刻んでスープに入れる。  
    → 把葱切碎放进汤里。

【ニンニク、ハム、野菜を〜】  
→ 切蒜、火腿、蔬菜

2. お墓に亡くなった人の名前が刻まれている。  
    → 墓碑上刻着逝者的名字。

【指輪、木、石、ガラスに〜】  
→ 刻在戒指、木头、石头、玻璃上

3. 長い歴史を刻んだ建物がそのまま残っている。  
    → 留有悠久历史痕迹的建筑仍然保留着。

【歴史、伝統、時を〜】  
→ 铭刻历史、传统、岁月

4. 彼の言葉が今も胸に深く刻まれている。  
    → 他的话至今仍深深铭刻在我心中。

【脳、心、記憶に〜】  
→ 铭刻在脑海、心里、记忆中

---

## 崩す（くずす）

### 含义：弄坏、打乱、搞垮

1. 形を崩さないようにケーキを切った。  
    → 小心切蛋糕，不让它变形。

【山、壁、豆腐を〜】  
→ 弄塌山、墙、豆腐

2. 列を崩さずまっすぐに並ぶ。  
    → 不打乱队形，排成一直线。

【バランス、調子、リズム、姿勢を〜】  
→ 打乱平衡、状态、节奏、姿势

3. 千円札を崩して全部百円玉にする。  
    → 把1000日元换成全部100日元硬币。

【お金を〜】  
→ 把大钱换开

---

## 狂う（くるう）

### 含义：发狂；失常；打乱

1. 仕事が忙しすぎて、ときどき気が狂いそうになる。  
    → 工作太忙，有时感觉快疯了。

【頭が〜】  
→ 脑子发疯

2. この時計は狂っているから、直してもらおう。  
    → 这块表不准了，拿去修吧。

【調子、リズム、感覚が〜】  
→ 状态、节奏、感觉失常

3. 急に友達が来たせいで、予定が狂ってしまった。  
    → 因为朋友突然来了，计划被打乱了。

【計画、人生、計算が〜】  
→ 计划、人生、计算出错

---

## 削る（けずる）

### 含义：削；削减

1. 鉛筆をナイフで削る。  
    → 用刀削铅笔。

【石、山、木、氷、骨、歯を〜】  
→ 削石头、山、木头、冰、骨头、牙齿

2. 睡眠時間を削って勉強する。  
    → 缩减睡眠时间学习。

【予算、費用、時間を〜】  
→ 削减预算、费用、时间

## 逆らう（さからう）

### 含义：违抗；逆流而行

1. 流れに逆らって川上に泳ぐ。  
    → 逆流向上游游去。

【風、重力、時代、流行に〜】  
→ 违逆风、重力、时代潮流、流行

2. 上司の命令に逆らったら、首になる。  
    → 如果违抗上司命令，就会被开除。

【指示、方針、神、親、先生に〜】  
→ 违抗指示、方针、神、父母、老师

---

## 縛る（しばる）

### 含义：绑；束缚

1. ごみは袋に入れ、袋の口をしっかり縛って出してください。  
    → 垃圾请装进袋子，并把袋口绑紧再拿出去。

【手足、髪、ひもを〜】  
→ 绑手脚、头发、绳子

2. 団体旅行は時間に縛られてしまう。  
    → 团体旅行会受到时间限制。

【規則、組織、常識に〜】  
→ 被规则、组织、常识束缚

---

## 搾る・絞る（しぼる）

### 含义：榨；拧干；集中

1. レモンを搾ってジュースを作る。  
    → 榨柠檬做果汁。

【果実、ジュース、牛乳を〜】  
→ 榨水果、果汁、牛奶

2. タオルはよく絞ってから干してください。  
    → 毛巾请拧干后再晾。

【ぞうきん、ハンカチを〜】  
→ 拧抹布、手帕

3. みんなで知恵を絞り、解決方法を考えた。  
    → 大家绞尽脑汁思考解决办法。

【アイデア、頭、声を〜】  
→ 绞尽点子、脑汁；压低声音

4. テーマを一つに絞ってレポートを書く。  
    → 把主题缩小到一个来写报告。

【焦点、的、人数、条件を〜】  
→ 缩小焦点、目标、人数、条件

## 染みる（しみる）

### 含义：渗入；触动

1. 雨がぬれて、雨水が地下まで染みてきた。  
    → 下雨后，雨水渗到了地下。

【汗、味、においが〜】  
→ 汗、水味、气味渗入

2. 冷たい水が虫歯に染みて痛い。  
    → 冷水刺激蛀牙，很疼。

【煙が目に〜、薬が傷に〜】  
→ 烟熏眼睛；药刺激伤口

3. つらいときほど父の優しさが心に染みる。  
    → 越是痛苦的时候，越能深深感受到父亲的温柔。

【胸に〜】  
→ 深深触动内心

---

## 迫る（せまる）

### 含义：逼近；迫使

1. 締め切り日後に迫っている。  
    → 截稿日期快到了。

【試験、選挙、危険が〜】  
→ 考试、选举、危险逼近

2. 次回この番組ではハワイの魅力に迫ります。  
    → 下次节目将深入介绍夏威夷的魅力。

【なぞ、素顔、問題に〜】  
→ 深入谜团、真面目、问题

3. 彼は彼女に結婚を迫られ、困ってしまった。  
    → 他被女方逼婚，很困扰。

【決断、選択を〜】  
→ 迫使作出决断、选择

## 注ぐ（そそぐ）

### 含义：注入；倾注

1. 関東平野を流れ、東京湾に注ぐ川。  
    → 流经关东平原、注入东京湾的河流。

【川が海に〜】  
→ 河流流入大海

2. カップにお湯を注ぐだけでスープができる。  
    → 只要往杯子里倒热水就能做成汤。

【酒、スープ、お茶を〜】  
→ 倒酒、汤、茶

3. 彼はこの十年間、研究に全力を注いできた。  
    → 他这十年来一直全力投入研究。

【力、愛情、エネルギーを〜】  
→ 倾注力量、爱情、精力

---

## 備える（そなえる）

### 含义：准备；具备

1. 地震に備えて水を買っておく。  
    → 为防地震先买好水。

【事故、火災、攻撃、トラブルに〜】  
→ 防备事故、火灾、攻击、麻烦

2. 最近の携帯電話は多くの機能を備えている。  
    → 现在的手机具备很多功能。

【性能、設備、施設を〜】  
→ 具备性能、设备、设施

3. 就職前に基本的な経済の知識を備えておく。  
    → 就业前先具备基本经济知识。

【魅力、技術、実力を〜】  
→ 具备魅力、技术、实力

## つかむ

### 含义：抓住；掌握；吸引

1. 警官は泥棒の腕をつかんで放さなかった。  
    → 警察抓住小偷的手臂不放。

【髪、ボール、ロープを〜】  
→ 抓头发、球、绳子

2. 苦労の末、彼はやっと幸せをつかんだ。  
    → 历经辛苦后，他终于抓住了幸福。

【チャンス、成功、情報、夢を〜】  
→ 抓住机会、成功、情报、梦想

3. 文章を読んで要点をつかむ。  
    → 阅读文章并掌握重点。

【コツ、ポイント、特徴、イメージを〜】  
→ 掌握诀窍、重点、特征、印象

4. 彼の歌声は一瞬で客の心をつかんだ。  
    → 他的歌声瞬间抓住了观众的心。

【気持ち、ファン、読者を〜】  
→ 抓住心情、粉丝、读者

## つぶす

### 含义：压碎；毁掉；消磨

1. イチゴをつぶしてジュースを作った。  
    → 把草莓压碎做成果汁。

【虫、缶、ジャガイモを〜】  
→ 压死虫子、压扁罐子、捣碎土豆

2. 父は無理な経営を続け、とうとう店をつぶしてしまった。  
    → 父亲因勉强经营，最终把店搞倒闭了。

【会社を〜】  
→ 搞垮公司

3. 子供の可能性をつぶしてはいけない。  
    → 不可以扼杀孩子的可能性。

【夢、チャンス、才能、計画を〜】  
→ 扼杀梦想、机会、才能、计划

4. 友達を待つ間、本を読んで時間をつぶした。  
    → 等朋友的时候看书打发时间。

【時間、暇、休みを〜】  
→ 消磨时间、空闲、假期
