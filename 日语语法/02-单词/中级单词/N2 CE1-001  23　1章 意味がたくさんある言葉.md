# 1章　意味がたくさんある言葉

## 2課　動詞②

## ウォーミングアップ

下面四个（　）中可以填入同一个动词。是什么词呢？

- まっすぐ行くと、通りに（　）
- 二台の車が（　）事故
- 親と（　）
- 壁に（　）

答案：**ぶつかる（碰撞、冲突）**

# 2 言葉

## 積む（つむ）

### 含义：堆积；装载；积累

1. 古新聞が山のように積んである。  
    → 旧报纸堆得像山一样。

【箱、野菜、木材を〜】  
→ 堆箱子、蔬菜、木材

2. トラックに引っ越しの荷物を積んだ。  
    → 把搬家的行李装上卡车。

【ものを車、船、バイク、自転車に〜】  
→ 把东西装到车、船、摩托车、自行车上

3. さまざまな体験を積んで人は成長する。  
    → 人通过积累各种经验而成长。

【経験、練習、トレーニングを〜】  
→ 积累经验、练习、训练

---

## 解く（とく）／溶く（とく）

### 含义：解开；解决；消除；调开

1. 引っ越しの荷物を解いて片付ける。  
    → 把搬家的行李拆开整理。

【包み、縄、ひも、包帯、帯を〜】  
→ 解开包裹、绳子、带子、绷带、腰带

2. この数学の問題は大学生でも解けないだろう。  
    → 这道数学题连大学生也解不开吧。

【なぞ、パズルを〜】  
→ 解谜、解拼图

3. 事情を説明して彼女の誤解を解いた。  
    → 说明情况后解除了她的误会。

【緊張、疑い、警戒を〜】  
→ 消除紧张、怀疑、戒备

4. 小麦粉を水と卵で溶いてケーキを作る。  
    → 用水和鸡蛋调开面粉做蛋糕。

【スープ、みそ、絵の具を〜】  
→ 调开汤、味噌、颜料

---

## 捕らえる（とらえる）

### 含义：捕捉；抓住；理解

1. 犯人は逃げようとしたが、すぐに警官に捕らえられた。  
    → 犯人想逃跑，但立刻被警察抓住了。

【魚、小鳥を〜】  
→ 捕捉鱼、小鸟

2. この小説は今も昔も多くの若者の心をとらえている。  
    → 这本小说无论过去还是现在都抓住了许多年轻人的心。

【心をとらえる】  
→ 抓住人心

3. 彼の描く人物画はよく特徴をとらえている。  
    → 他画的人物画很好地抓住了人物特征。

【変化、状況、意味を〜】  
→ 抓住变化、状况、意义

4. 失敗は成功するためのチャンスととらえることもできる。  
    → 失败也可以被看作成功的机会。

---

## 握る（にぎる）

### 含义：握；掌握；抓住把柄

1. 子供は母親の手をしっかり握った。  
    → 孩子紧紧握住母亲的手。

【手首、ロープ、小石を〜】  
→ 握住手腕、绳子、小石子

2. 彼女はハンドルを握ると、性格が変わる。  
    → 她一握方向盘，性格就变了。

【マイク、ペン、包丁を〜】  
→ 握麦克风、笔、菜刀

3. 我が家は母が財布を握っている。  
    → 我家是母亲掌管财政。

【権力、力、事件のかぎを〜】  
→ 掌握权力、力量、事件关键

4. 彼に弱みを握られているので強く言えない。  
    → 因为把柄被他抓住了，所以不敢强硬地说。

【秘密、弱点を〜】  
→ 抓住秘密、弱点

---

## にらむ

### 含义：瞪；仔细看；推测

1. 怒られた子供は母親の顔をじっとにらんでいた。  
    → 被骂的孩子一直瞪着母亲的脸。

【こっち、（人）を〜】  
→ 瞪这边、某人

2. 彼は真剣な顔でデータをにらんでいる。  
    → 他表情严肃地盯着数据。

【試験問題、書類、鏡を〜】  
→ 盯着试题、文件、镜子

3. 警察は彼を犯人とにらんでいる。  
    → 警察怀疑他是犯人。

---

## 跳ねる（はねる）

### 含义：跳跃；飞溅；翘起；撞飞

1. 網の中でたくさんの魚が跳ねている。  
    → 网里很多鱼在乱跳。

【ウサギ、ボールが〜】  
→ 兔子、球跳动

2. 車が横を通ったとき、スカートに泥がはねた。  
    → 车从旁边经过时，泥水溅到了裙子上。

【油、水が〜】  
→ 油、水飞溅

3. 髪の毛がはねているけど、寝坊したの？  
    → 头发翘起来了，你睡过头了吗？

【ひげが〜】  
→ 胡子翘起

4. はねられる  
    友人がタクシーにはねられて、入院した。  
    → 朋友被出租车撞到住院了。

【人、動物が車、バイクに〜】  
→ 人、动物被车或摩托撞

---

## はまる

### 含义：嵌入；陷入；沉迷；符合

1. ボタンが大きすぎてなかなかはまらない。  
    → 扣子太大，很难扣进去。

【指輪、ふた、ねじが〜】  
→ 戒指、盖子、螺丝卡住

2. 急いでいるのに、バスが渋滞にはまってしまった。  
    → 明明很赶时间，公交车却堵车了。

【わな、深みに〜】  
→ 陷入陷阱、泥潭

3. 兄はゲームにはまっていて、夜もあまり寝ずにやっている。  
    → 哥哥沉迷游戏，晚上也不怎么睡觉。

【映画に〜】  
→ 沉迷电影

4. 型にはまらず、自由な発想で考えてください。  
    → 不要拘泥于固定模式，自由思考。

【枠、パターンに〜】  
→ 陷入框架、模式

# 第95页

## 膨らむ（ふくらむ）

### 含义：膨胀；增加；充满期待

1. 風船が大きく膨らんだ。  
    → 气球鼓了起来。

【財布、おなか、ポケットが〜】  
→ 钱包、肚子、口袋鼓起来

2. A選手の今後の活躍に期待が膨らむ。  
    → 对A选手今后的表现充满期待。

【イメージ、想像、夢が〜】  
→ 想象、梦想扩大

3. 借金がどんどん膨らんでしまって、返せない。  
    → 债务越来越大，已经还不上了。

【予算、赤字が〜】  
→ 预算、赤字扩大

---

## ふさがる

### 含义：堵住；愈合；占满

1. 台風で木が倒れて、道がふさがっている。  
    → 因台风树倒了，道路被堵住了。

【道路、入口、血管が〜】  
→ 道路、入口、血管堵塞

2. 一週間たって、やっと傷がふさがった。  
    → 过了一周，伤口终于愈合了。

【穴、隙間が〜】  
→ 洞、缝隙被堵住

3. 荷物で手がふさがっていて、ドアが開けられない。  
    → 手里拿满了东西，打不开门。

【部屋、席、予定が〜】  
→ 房间、座位、日程被占满

---

## ぶつかる

### 含义：碰撞；冲突；遇到困难

1. トラックとバスがぶつかった。  
    → 卡车和公交车相撞了。

荷物が人にぶつかってしまった。  
→ 行李撞到了人。

2. この道をまっすぐ行くと大通りにぶつかる。  
    → 这条路一直走会碰到大马路。

【広い道、T字路に〜】  
→ 碰到大路、T字路

3. 試験日と卒業式がぶつかってしまった。  
    → 考试日和毕业典礼撞期了。

【予定、日程、スケジュールが〜】  
→ 行程、日程、计划冲突

4. 壁にぶつかって、研究が進まない。  
    → 遇到瓶颈，研究无法推进。

【困難、疑問、問題に〜】  
→ 遇到困难、疑问、问题

5. 上司と意見がぶつかって、会社を辞めた。  
    → 和上司意见冲突后辞职了。

【人と意見が〜】  
→ 与人意见冲突

---

## 振る（ふる）

### 含义：挥动；撒；甩掉；注音

1. 犬はうれしいとしっぽを振る。  
    → 狗高兴时会摇尾巴。

【手、首、瓶、旗を〜】  
→ 挥手、摇头、摇瓶子、挥旗

2. 野菜が煮えたら、器に入れ、最後にごまを振る。  
    → 菜煮好后装盘，最后撒上芝麻。

【塩、こしょうを〜】  
→ 撒盐、胡椒

3. 彼女に告白したら振られてしまった。  
    → 向她告白后被甩了。

【人を〜】  
→ 甩掉别人

4. 名簿の名前にふりがなを振る。  
    → 给名单上的名字标注假名。

【番号、読みがなを〜】  
→ 标注号码、读音

---

## 触れる（ふれる）

### 含义：触碰；接触；涉及；违反

1. 作品に手を触れないでください。  
    → 请不要触摸作品。

前に立っている人の髪が顔に触れる。  
→ 前面人的头发碰到了脸。

2. この金属は空気に触れると黒くなる。  
    → 这种金属接触空气会变黑。

【光、熱に〜】  
→ 接触光、热

3. 海外旅行は外国の文化に触れられるいい機会だ。  
    → 出国旅行是接触外国文化的好机会。

【芸術、魅力、優しさに〜】  
→ 接触艺术、魅力、温柔

4. 彼女と話すときは、年齢には触れないほうがいい。  
    → 和她说话时最好别提年龄。

【話題、問題に〜】  
→ 提及话题、问题

5. 日本では二十歳以下の者の喫煙は法律に触れる。  
    → 在日本，未满20岁吸烟违法。

【法、規則に〜】  
→ 触犯法律、规则

6. 貴重品は人目に触れないところに置く。  
    → 贵重物品放在人看不到的地方。

【目、耳に〜】  
→ 被看到、被听到

---

## 招く（まねく）

### 含义：招待；招致

1. 結婚式に大学時代の友人を招いた。  
    → 婚礼邀请了大学时代的朋友。

【食事、パーティー、式、家に〜】  
→ 邀请吃饭、参加派对、典礼、来家里

2. 説明が不十分で誤解を招いてしまった。  
    → 因说明不足而引起了误会。

【危険、事故、混乱を〜】  
→ 招致危险、事故、混乱

---

## 寄せる（よせる）

### 含义：靠近；寄来；集中

1. 車を右に寄せて止める。  
    → 把车靠右停下。

【端、片側、中央、一か所に〜】  
→ 靠到边上、一侧、中央、某处

2. 詳細に対するご意見、ご感想をお寄せください。  
    → 请提供对详细内容的意见和感想。

【便り、情報、苦情を〜】  
→ 寄来消息、信息、投诉

3. この現象に多くの学者が関心を寄せている。  
    → 许多学者对这一现象表示关注。

【期待、思い、信頼を〜】  
→ 寄予期待、感情、信赖
