# 2　言葉（词汇）

## 1．数量・程度（数量・程度）

① この野菜はビタミンを＿＿＿含む

- 豊富に・たっぷり  
    → 这种蔬菜富含维生素

② 日本は天然資源が乏しい  
→ 日本天然资源贫乏

③ ここでは作物がわずかしか取れない  
→ 这里农作物收成很少

④ 大量の水を飲む  
→ 喝大量的水

⑤ 微量の薬を混ぜる  
→ 混入少量药物

⑥ 料理に塩を少々加える  
→ 做菜时加一点盐

⑦ 大規模な調査をする  
→ 进行大规模调查

⑧ 高度な技術・テクニックを持つ  
→ 拥有高超技术

⑨ これはまれに起こる問題だ  
→ 这是很少发生的问题

⑩ 風邪気味で少し熱がある  
→ 有点感冒，稍微发烧

⑪ 苦の経験が大いに役立つ  
→ 苦涩的经验非常有帮助

⑫ 政府の最大の課題は景気回復だ  
→ 政府最大的课题是恢复经济

⑬ 今日の最低気温は1度だった  
→ 今天最低气温是1度

⑭ 日中の最高気温は二十度に達した  
→ 白天最高气温达到20度

⑮ ぶかぶかの服を着る  
→ 穿宽松肥大的衣服

⑯ ＿＿＿を測定する

- 幅・間隔
- 距離  
    → 测量宽度／间隔／距离

⑰ 立体（立方体・球）の体積  
→ 立体（立方体、球体）的体积

⑱ 図形（長方形・正方形・円）の面積  
→ 图形（长方形、正方形、圆形）的面积

⑲ 単位（单位）

- cm：センチメートル → 厘米
- cm²：平方センチメートル → 平方厘米
- cm³：立方センチメートル → 立方厘米
- ℓ：リットル → 升
- t：トン → 吨

## 2．増減・過不足（增减・过剩不足）

### ▶ 計算（计算）

①

- 足し算：2＋3＝5（2足す3は5）→ 加法
- 引き算：5－2＝3（5引く2は3）→ 减法
- 掛け算：2×3＝6（2掛ける3は6）→ 乘法
- 割り算：6÷2＝3（6割る2は3）→ 除法

② 甘みが足りなかったので砂糖を足した  
→ 因为甜味不够，所以加了糖

③ 給料から税金が引かれる  
→ 税金从工资中扣除

④ 勘定を人数で割る→割り勘にする  
→ 按人数平摊费用

⑤ 金額を計算する  
→ 计算金额

⑥ 人口が増加する→減少する  
→ 人口增加／减少

⑦ 会社の規模を縮小する  
→ 缩小公司规模

⑧ 小麦の需給が増大する→低下する  
→ 小麦供需扩大／下降

⑨ チームに新しいメンバーが加わる  
→ 队伍加入了新成员

⑩ 仕事の経験を積む  
→ 积累工作经验

⑪ 不足分を補う  
→ 补足不足部分

⑫ 負担を軽減する  
→ 减轻负担

大幅　　　　　比普通幅面的布
多少　　　　　指数量或多或少

ずらりと　　　一大排

**(1) 東京ドームの直径は約244メートルで、面積はおよそ4万7千平方メートルです。**  
**中文解释：** 东京巨蛋的直径约244米，面积约为4万7千平方米。

- 「直径」：直径
    
- 「面積」：面积
    
- 「およそ」：大约
    

---

**(2) 我が国の経済の一層の発展のためには、輸出を増大させるよりほかにない。**  
**中文解释：** 为了我国经济的进一步发展，除了扩大出口之外别无他法。

- 「一層の発展」：进一步发展
    
- 「～よりほかにない」：除～之外没有别的办法
    

---

**(3) この地区での犯罪の発生件数は2007年をピークに大幅に減少している。**  
**中文解释：** 该地区的犯罪发生件数以2007年为峰值，之后大幅减少。

- 「～をピークに」：以～为顶峰
    
- 「大幅に減少」：大幅度减少
    

---

**(4) 山田さんは、まだ入社一年目だから、多少経験不足なのは仕方がない。**  
**中文解释：** 山田先生（小姐）因为才进公司第一年，所以多少有点经验不足也是没办法的事。

- 「入社一年目」：进公司第一年
    
- 「経験不足」：经验不足
    
- 「仕方がない」：无可奈何
    

---

**(5) 箱の中にはおいしそうなお菓子がぎっしり詰まっている。**  
**中文解释：** 箱子里塞满了看起来很好吃的点心。

- 「おいしそうな」：看起来好吃的
    
- 「ぎっしり」：满满地
    
- 「詰まっている」：塞满着
    

---

**(6) 本屋の棚には本や雑誌がずらりと並んでいる。**  
**中文解释：** 书店的书架上摆满了一排排的书和杂志。

- 「ずらりと」：一排排地、成排地
    
- 「並んでいる」：陈列着
    

---

**(7) そろそろ帰る時間かと思い、時計をちょっと見た。**  
**中文解释：** 想着差不多到该回去的时间了吧，就稍微看了一下表。

- 「そろそろ」：差不多要～
    
- 「～かと思い」：心想是否～
    
- 「ちょっと見た」：看了一下
