### 促音、长音、拨音

#### 促音：小 `っ / ッ`

读法：停顿一拍，后面的辅音加强。

例词：

- きって：切手，邮票
- がっこう：学校，学校
- ざっし：雑誌，杂志

#### 长音

读法：把前面的音拉长一拍。

常见形式：

- `あ段 + あ`：おかあさん
- `い段 + い`：おにいさん
- `う段 + う`：くうこう
- `え段 + い / え`：せんせい、おねえさん
- `お段 + う / お`：おとうさん、おおきい
- 片假名常用 `ー`：コーヒー、スーパー

#### 拨音：`ん / ン`

读法：鼻音，后面接不同音时实际发音会轻微变化。

例词：

- ほん：本，书
- しんぶん：新聞，报纸
- でんしゃ：電車，电车

### 平假名和片假名使用场景

- 平假名：日语固有词、助词、动词和形容词活用词尾。
- 片假名：外来语、外国人名、外国地名、拟声拟态词、强调。
- 汉字：名词、动词词干、形容词词干等。

例：

- わたしはパンを食べます。
- コーヒーを飲みます。
- 日本語を勉強します。

## 3. N5 语法

### 3.1 基础必会语法

### 判断句

- `名词 + です`：是……
  - 私は学生です。
- `名词 + ではありません / じゃありません`：不是……
  - 田中さんは先生ではありません。
- `名词 + でした`：过去是……
  - 昨日は休みでした。
- `名词 + ではありませんでした`：过去不是……
  - 昨日は雨ではありませんでした。
- `名词 + ですか`：是……吗？
  - あなたは学生ですか。

### 指示词

- `これ / それ / あれ / どれ`：这个 / 那个 / 那个 / 哪个
- `この / その / あの / どの + 名词`：这个…… / 那个…… / 哪个……
- `ここ / そこ / あそこ / どこ`：这里 / 那里 / 那边 / 哪里
- `こちら / そちら / あちら / どちら`：这里 / 那里 / 那边 / 哪边；也可表示人或方向
- `こんな / そんな / あんな / どんな + 名词`：这样的 / 那样的 / 怎样的

### 存在句

- `名词があります`：有……，用于无生命物
  - 机の上に本があります。
- `名词がいます`：有……，用于人和动物
  - 教室に学生がいます。
- `地点に 名词があります / います`：某地有……
- `名词は 地点にあります / います`：某物 / 某人在某地

### 动词基本形和ます形

- `动词ます`：现在、将来、习惯动作
  - 毎日勉強します。
- `动词ません`：否定
  - 肉を食べません。
- `动词ました`：过去
  - 昨日映画を見ました。
- `动词ませんでした`：过去否定
  - 昨日勉強しませんでした。
- `动词ましょう`：一起……吧
  - 一緒に帰りましょう。
- `动词ませんか`：邀请
  - コーヒーを飲みませんか。

### 动词て形

- `动词てください`：请……
  - ここに名前を書いてください。
- `动词てもいいです`：可以……
  - 写真を撮ってもいいです。
- `动词てはいけません`：不可以……
  - ここでタバコを吸ってはいけません。
- `动词ています`：正在……；状态持续；习惯职业
  - 今、本を読んでいます。
- `动词てから`：……之后
  - ご飯を食べてから、勉強します。
- `动词て、动词て`：动作连续
  - 朝起きて、顔を洗って、ご飯を食べます。

### 动词ない形

- `动词ないでください`：请不要……
  - ここで写真を撮らないでください。
- `动词なければなりません`：必须……
  - 宿題をしなければなりません。
- `动词なくてもいいです`：不……也可以
  - 明日来なくてもいいです。

### 动词た形

- `动词たことがあります`：曾经……
  - 日本へ行ったことがあります。
- `动词たり、动词たりします`：列举动作
  - 休みの日、映画を見たり、音楽を聞いたりします。
- `动词た後で`：……之后
  - 授業が終わった後で、図書館へ行きます。

### 形容词

- `い形容词 + です`：……的
  - この本はおもしろいです。
- `い形容词くないです`：不……
  - 今日は寒くないです。
- `い形容词かったです`：过去……
  - 昨日は暑かったです。
- `い形容词くなかったです`：过去不……
  - 昨日は寒くなかったです。
- `な形容词 + です`：……的
  - この町は静かです。
- `な形容词ではありません`：不……
  - この部屋はきれいではありません。
- `な形容词でした`：过去……
  - 昨日は暇でした。
- `な形容词ではありませんでした`：过去不……
  - 昨日は暇ではありませんでした。
- `い形容词くて`：又……又……
  - この店は安くておいしいです。
- `な形容词で`：又……又……
  - この町は静かで便利です。

### 比较

- `AはBより + 形容词です`：A 比 B 更……
  - 中国は日本より広いです。
- `AよりBのほうが + 形容词です`：比起 A，B 更……
  - バスより電車のほうが速いです。
- `AとBとどちらが + 形容词ですか`：A 和 B 哪个更……
  - 肉と魚とどちらが好きですか。
- `Aの中でBがいちばん + 形容词です`：在 A 中 B 最……
  - 季節の中で春がいちばん好きです。

### 愿望、能力、喜好

- `名词が好きです`：喜欢……
  - 音楽が好きです。
- `名词が嫌いです`：讨厌……
  - 野菜が嫌いです。
- `名词が上手です`：擅长……
  - 田中さんは料理が上手です。
- `名词が下手です`：不擅长……
  - 私は歌が下手です。
- `名词がわかります`：懂……
  - 日本語が少しわかります。
- `名词ができます`：会……；能……
  - テニスができます。
- `动词ます形去ます + たいです`：想……
  - 日本へ行きたいです。

### 原因、目的、转折

- `から`：因为……
  - 暑いですから、窓を開けます。
- `ので`：因为……，较柔和
  - 雨なので、行きません。
- `でも`：但是
  - 日本語は難しいです。でも、おもしろいです。
- `が`：但是
  - 安いですが、あまりよくないです。
- `动词ます形去ます + に行きます / 来ます / 帰ります`：去 / 来 / 回来做……
  - 本を買いに行きます。

### 时间和频率

- `もう + 动词ました`：已经……
  - もう昼ご飯を食べました。
- `まだ + 动词ていません`：还没有……
  - まだ宿題をしていません。
- `いつも`：总是
- `よく`：经常
- `時々`：有时
- `あまり + 否定`：不太……
- `全然 + 否定`：完全不……

### 连接和名词化

- `动词普通形 + こと`：把动作名词化
  - 日本語を勉強することは楽しいです。
- `动词普通形 + の`：把动作名词化
  - 本を読むのが好きです。
- `名词 / な形容词 + な + こと`
- `い形容词 + こと`

### 请求、建议、许可

- `动词たほうがいいです`：最好……
  - 早く寝たほうがいいです。
- `动词ないほうがいいです`：最好不要……
  - お酒を飲まないほうがいいです。
- `动词ましょうか`：我来……吧？
  - 手伝いましょうか。
- `动词てもいいですか`：可以……吗？
  - 入ってもいいですか。

### 3.2 追加必背语法

### 动词辞书形 + 前に

意思：在……之前。

- 寝る前に、歯を磨きます。
- ご飯を食べる前に、手を洗います。

### 名词 + の前に

意思：在……之前；在……前面。

- 試験の前に、勉強します。
- 駅の前にコンビニがあります。

### 动词た形 + ことがある

意思：曾经做过……。

- 日本へ行ったことがあります。
- すしを食べたことがあります。

### 动词た形 + り、动词た形 + りする

意思：列举几个动作。

- 休みの日、映画を見たり、買い物したりします。
- 公園で走ったり、写真を撮ったりしました。

### 动词普通形 / い形容词 / な形容词だ / 名词だ + と思います

意思：我认为……；我觉得……。

- 明日は雨だと思います。
- 日本語はおもしろいと思います。

### 普通形 + と言います

意思：说……。

- 先生は明日テストがあると言いました。
- 彼は行かないと言いました。

### 名词 + がほしいです

意思：想要……。

- 新しいパソコンがほしいです。
- 水がほしいです。

### 动词ます形去ます + たいです

意思：想做……。

- 日本へ行きたいです。
- 冷たい水を飲みたいです。

### 动词ます形去ます + ながら

意思：一边……一边……。

- 音楽を聞きながら、勉強します。
- テレビを見ながら、ご飯を食べます。

### 动词て形 + いる

意思：正在……；状态持续。

- 今、本を読んでいます。
- 田中さんは結婚しています。

### 动词て形 + しまう

意思：做完；遗憾地……。

- 宿題を全部やってしまいました。
- 財布を忘れてしまいました。

### 名词 / な形容词 + になる、い形容词く + なる

意思：变成……。

- 先生になりたいです。
- 寒くなりました。
- この町は静かになりました。

### 名词 / な形容词 + にする、い形容词く + する

意思：使……变成……；决定选择……。

- 部屋をきれいにします。
- 音を小さくしてください。
- コーヒーにします。

### 动词普通形 + ことができます

意思：能够做……。

- 日本語を話すことができます。
- ここで写真を撮ることができます。

### 动词ない形 + ほうがいいです

意思：最好不要……。

- 夜遅く食べないほうがいいです。
- 今日は出かけないほうがいいです。

### 动词た形 + ほうがいいです

意思：最好……。

- 早く寝たほうがいいです。
- 病院へ行ったほうがいいです。

### 名词 + だけ

意思：只有……；仅仅……。

- 水だけ飲みました。
- 教室に学生だけいます。

### 名词 + しか + 否定

意思：只有……，语气比 `だけ` 更强调少。

- 100円しかありません。
- 日本語は少ししかわかりません。

### 名词 + か、名词 + か

意思：……或者……。

- コーヒーかお茶を飲みます。
- 土曜日か日曜日に行きます。

### 疑问词 + か

意思：某……；不确定。

- 何か食べたいです。
- どこかへ行きたいです。

### 疑问词 + も + 否定

意思：全面否定。

- 何も食べませんでした。
- 誰もいません。
- どこへも行きません。

## 4. N5 助词分类和使用场景

### 4.1 按助词类别分类

| 助词类别 | N5 常见助词 | 核心作用              | 使用场景              | 例句               |
| ---- | ------- | ----------------- | ----------------- | ---------------- |
| 格助词  | `が`     | 标记主语、存在对象、能力对象    | 表示“谁/什么”是动作或状态主体  | 雨が降っています。        |
| 格助词  | `を`     | 标记动作对象            | 他动词前的宾语           | パンを食べます。         |
| 格助词  | `に`     | 标记时间点、存在地点、到达点、对象 | 时间、存在、移动目的地、动作对象  | 7時に起きます。         |
| 格助词  | `へ`     | 标记方向              | 和 `行く、来る、帰る` 连用   | 学校へ行きます。         |
| 格助词  | `で`     | 标记动作地点、工具、手段、范围   | 在某地做事、用某工具、乘坐交通工具 | 図書館で勉強します。       |
| 格助词  | `と`     | 标记共同动作对象、引用、完全列举  | 和某人一起；引用说话内容      | 友達と映画を見ます。       |
| 格助词  | `から`    | 标记起点、原因           | 从某时间/地点开始；说明理由    | 9時から勉強します。       |
| 格助词  | `まで`    | 标记终点              | 到某时间/地点为止         | 5時まで働きます。        |
| 格助词  | `より`    | 标记比较基准            | A 比 B 更……         | 中国は日本より広いです。     |
| 格助词  | `の`     | 标记所属、修饰、同位        | A 的 B；名词修饰名词      | 私の本です。           |
| 提示助词 | `は`     | 提示主题、对比           | 说明“关于……”，或对比两个主题  | 私は中国人です。         |
| 提示助词 | `も`     | 表示“也”、同类追加        | 追加同类信息；全面否定搭配     | 私も行きます。          |
| 副助词  | `だけ`    | 表示限定              | 只有……              | 水だけ飲みました。        |
| 副助词  | `しか`    | 表示限定，后接否定         | 只有……，强调少          | 100円しかありません。     |
| 副助词  | `か`     | 表示不确定、选择          | 某……；A 或 B         | 何か食べたいです。        |
| 并列助词 | `と`     | 完全列举              | A 和 B，列举对象全部明确    | 本とノートがあります。      |
| 并列助词 | `や`     | 不完全列举             | A 啦 B 啦，还有其他      | 本やノートがあります。      |
| 接续助词 | `から`    | 表示原因              | 因为……所以……          | 暑いですから、窓を開けます。   |
| 接续助词 | `が`     | 表示转折、铺垫           | 虽然……但是……          | 安いですが、あまりよくないです。 |
| 接续助词 | `ので`    | 表示原因              | 因为……，语气比 `から` 柔和  | 雨なので、行きません。      |
| 终助词  | `か`     | 表示疑问              | 放句尾构成疑问句          | 学生ですか。           |
| 终助词  | `ね`     | 表示确认、共鸣           | 和对方确认或寻求认同        | いい天気ですね。         |
| 终助词  | `よ`     | 表示提醒、告知           | 告诉对方不知道的信息        | 明日は試験ですよ。        |

记忆重点：

- 格助词主要负责说明名词和谓语之间的关系，例如主语、对象、地点、方向、时间。
- 提示助词主要负责把某个内容提出来作为话题，例如 `は`、`も`。
- 副助词会给词语增加“限定、程度、不确定”等附加意义，例如 `だけ`、`しか`、`か`。
- 接续助词连接两个句子或两个小句，例如原因、转折。
- 终助词放在句尾，表达说话人的语气，例如疑问、确认、提醒。

### 4.2 按使用场景分类

### 主题、主语、对象类

#### は

使用场景：

- 提示句子的主题，说明“关于……”
- 对比两个事物
- 介绍自己、身份、职业、国籍

例句：

- 私は中国人です。
- 田中さんは先生です。
- コーヒーは飲みますが、お酒は飲みません。

注意：

- `は` 后面常接对这个主题的说明。
- 读音是 `わ`，不是 `は`。

#### が

使用场景：

- 提示主语，强调“谁 / 什么”做动作或处于某状态
- 表示存在句中的存在对象
- 表示能力、喜好、懂不懂的对象
- 疑问词作主语时，后面通常用 `が`

例句：

- 雨が降っています。
- 教室に学生がいます。
- 私は日本語が少しわかります。
- 誰が来ましたか。

注意：

- `は` 强调主题，`が` 强调主语或新信息。
- `好き、嫌い、上手、下手、わかる、できる` 前面的对象常用 `が`。

#### を

使用场景：

- 表示他动词的动作对象
- 表示经过、离开的地点，N5 阶段先重点掌握动作对象

例句：

- パンを食べます。
- 日本語を勉強します。
- 本を読みます。
- 公園を散歩します。

注意：

- 读音是 `お`，不是 `を`。

### 地点、方向、时间类

#### に

使用场景：

- 表示具体时间点
- 表示存在地点
- 表示动作到达点、目的地
- 表示动作对象
- 表示目的，常用 `动词ます形去ます + に行きます`

例句：

- 7時に起きます。
- 机の上に本があります。
- 学校に行きます。
- 友達に電話します。
- 本を買いに行きます。

注意：

- `今日、明日、毎日` 这类相对时间通常不加 `に`。
- 有明确数字的时间常加 `に`，如 `7時に`、`月曜日に`。

#### へ

使用场景：

- 表示移动方向，重点是“朝……方向”
- 常和 `行く、来る、帰る` 一起使用

例句：

- 学校へ行きます。
- 日本へ来ました。
- 家へ帰ります。

注意：

- 读音是 `え`，不是 `へ`。
- `に` 强调到达点，`へ` 强调方向。N5 中很多移动句两者都可以用。

#### で

使用场景：

- 表示动作发生的地点
- 表示交通工具、工具、手段
- 表示范围
- 表示材料或语言等方式

例句：

- 図書館で勉強します。
- 電車で学校へ行きます。
- はしでご飯を食べます。
- 日本語で話します。
- クラスで田中さんがいちばん背が高いです。

注意：

- `に` 表示存在地点，`で` 表示动作地点。
- `部屋にいます` 是“在房间里”；`部屋で勉強します` 是“在房间学习”。

### 所属、并列、共同动作类

#### の

使用场景：

- 表示所属
- 名词修饰名词
- 代替前面出现过的名词
- 表示产地、内容、性质

例句：

- これは私の本です。
- 日本語の先生です。
- 赤いかばんは私のです。
- 日本の車です。

注意：

- `AのB` 基本可以理解为“A 的 B”或“A 类型的 B”。

#### と

使用场景：

- 表示“和……”
- 表示共同动作对象
- 表示完全列举
- 表示引用，说话内容后接 `言います`

例句：

- 友達と映画を見ます。
- 机の上に本とノートがあります。
- 母と電話しました。
- 先生は「休んでください」と言いました。

注意：

- `と` 是全部列举，`や` 是部分列举。

#### や

使用场景：

- 表示不完全列举，意思是“……啦……啦”
- 暗示还有其他同类事物

例句：

- 机の上に本やノートがあります。
- スーパーで野菜や果物を買いました。

注意：

- `AとB` 表示只有 A 和 B。
- `AやB` 表示举例 A、B，还有其他。

### 添加、起点、终点、比较类

#### も

使用场景：

- 表示“也”
- 表示同类追加
- 和疑问词、否定搭配，表示全面否定

例句：

- 私も行きます。
- 田中さんも学生です。
- 何も食べませんでした。
- どこへも行きません。

注意：

- `も` 会替代 `は、が、を`，例如 `私も`、`水も飲みます`。

#### から

使用场景：

- 表示时间或地点的起点：从……
- 表示原因：因为……
- 表示材料来源，N5 阶段先掌握起点和原因

例句：

- 9時から勉強します。
- 駅から学校まで歩きます。
- 暑いですから、窓を開けます。

注意：

- 表示原因时语气直接，常用于口语。

#### まで

使用场景：

- 表示时间或地点的终点：到……
- 常和 `から` 搭配使用

例句：

- 5時まで働きます。
- 家から駅まで歩きます。
- 月曜日から金曜日まで学校へ行きます。

注意：

- `から` 是起点，`まで` 是终点。

#### より

使用场景：

- 表示比较基准：比……
- 常用于 `AはBより...です`

例句：

- 中国は日本より広いです。
- 電車はバスより速いです。

注意：

- `より` 后面的 B 是被比较对象。
- `AよりBのほうが...` 表示“比起 A，B 更……”。

### 句尾语气类

#### ね

使用场景：

- 表示确认
- 表示希望对方同意
- 表示共鸣、感叹

例句：

- いい天気ですね。
- この店はおいしいですね。
- 明日は休みですね。

注意：

- `ね` 的语气柔和，常用于和对方共享信息或确认。

#### よ

使用场景：

- 告诉对方不知道的信息
- 提醒、强调
- 给建议或轻微警告

例句：

- この本はおもしろいですよ。
- 明日は試験ですよ。
- 早く行ったほうがいいですよ。

注意：

- `よ` 比 `ね` 更强调“我告诉你”。
- `ですよね` 可以表示“是吧、对吧”，兼有告知和确认语气。

### N5 助词高频对比

| 对比          | 区别                    | 例句               |
| ----------- | --------------------- | ---------------- |
| `は` / `が`   | `は` 提示主题，`が` 强调主语或新信息 | 私は学生です。誰が来ましたか。  |
| `に` / `で`   | `に` 表示存在地点，`で` 表示动作地点 | 部屋にいます。部屋で勉強します。 |
| `に` / `へ`   | `に` 强调到达点，`へ` 强调方向    | 学校に行きます。学校へ行きます。 |

### 动词

动词词性说明：

- `一类动词 / 五段动词`：词尾多为 `う、く、ぐ、す、つ、ぬ、ぶ、む、る`，ます形按五段变化。
- `二类动词 / 一段动词`：多以 `いる、える` 结尾，ます形去 `る` 加 `ます`。
- `三类动词 / 不规则动词`：`する`、`来る` 以及 `名词 + する`。
- `自动词`：动作或变化自然发生，重点在主语本身，常用 `が`。
- `他动词`：人为作用到对象上，常带宾语，常用 `を`。

### 5.3 动词归类记忆

#### 按动词词性归类

| 动词词性 | 记忆规则 | N5 常见动词 |
|---|---|---|
| 一类 / 五段动词 | 词尾多为 `う、く、ぐ、す、つ、ぶ、む、る`，ます形词尾变 `い段 + ます` | 行く、帰る、飲む、読む、書く、話す、買う、待つ、持つ、休む、遊ぶ、泳ぐ、歩く、走る、分かる |
| 二类 / 一段动词 | 多以 `いる、える` 结尾，ます形去 `る + ます` | 起きる、寝る、食べる、見る、出る、開ける、閉める、借りる、教える、忘れる、覚える |
| 三类 / サ变动词 | `する` 或 `名词 + する` | 勉強する、仕事する、掃除する、洗濯する、料理する、運転する、電話する、散歩する |
| 三类 / カ变动词 | 只有 `来る` | 来る |

#### 按自动词、他动词归类

| 类型 | 记忆重点 | N5 常见动词 | 常用助词 |
|---|---|---|---|
| 自动词 | 主语自己动作或状态变化，重点在“谁/什么发生变化” | 行く、来る、帰る、起きる、寝る、入る、出る、座る、立つ、遊ぶ、泳ぐ、歩く、走る、分かる、ある、いる | `が`、`に`、`へ`、`で` |

| 自动词 / 他动词 | 根据句子对象不同，两种用法都可能出现 | 休む、仕事する、料理する | `を`、`で` |

## 6. N5 必须背诵的文章

### 6.1 基础背诵文章

### 文章 1：自我介绍

はじめまして。私は王明です。中国から来ました。今、日本語を勉強しています。私は会社員です。毎日、朝七時に起きます。八時に会社へ行きます。日本語は少し難しいですが、とてもおもしろいです。どうぞよろしくお願いします。

中文：

初次见面。我是王明。我来自中国。现在正在学习日语。我是公司职员。每天早上七点起床。八点去公司。日语有点难，但是非常有趣。请多关照。

### 文章 2：我的一天

私は毎日六時半に起きます。顔を洗って、朝ご飯を食べます。七時半に家を出ます。電車で学校へ行きます。午前中、日本語を勉強します。昼休みに友達とご飯を食べます。午後、図書館で本を読みます。夜、家へ帰って、宿題をします。十一時ごろ寝ます。

中文：

我每天六点半起床。洗脸，然后吃早饭。七点半出门。坐电车去学校。上午学习日语。午休时和朋友一起吃饭。下午在图书馆看书。晚上回家后做作业。十一点左右睡觉。

### 文章 3：周末

土曜日と日曜日は休みです。土曜日の朝、部屋を掃除します。それから、スーパーへ行って、食べ物を買います。午後、友達と映画を見ます。日曜日はあまり出かけません。家で音楽を聞いたり、本を読んだりします。週末はとても楽しいです。

中文：

星期六和星期天休息。星期六早上打扫房间。然后去超市买食物。下午和朋友看电影。星期天不太出门。在家听音乐、看书。周末非常开心。

### 文章 4：买东西

昨日、デパートへ行きました。新しいかばんを買いたかったです。店に赤いかばんと黒いかばんがありました。赤いかばんはきれいでしたが、高かったです。黒いかばんは安くて、軽かったです。私は黒いかばんを買いました。

中文：

昨天去了百货商店。我想买一个新包。店里有红色的包和黑色的包。红色的包很漂亮，但是很贵。黑色的包便宜又轻。我买了黑色的包。

### 文章 5：我的家

私の家は駅の近くにあります。家の前に小さい公園があります。家族は四人です。父と母と兄と私です。父は医者です。母は先生です。兄は大学生です。私の部屋はあまり大きくないですが、明るくてきれいです。机の上に本とパソコンがあります。

中文：

我家在车站附近。家前面有一个小公园。家里有四个人。爸爸、妈妈、哥哥和我。爸爸是医生。妈妈是老师。哥哥是大学生。我的房间不太大，但是明亮又干净。桌子上有书和电脑。

### 文章 6：天气

今日はいい天気です。朝は少し寒かったですが、昼は暖かくなりました。私は友達と公園へ行きました。公園には花がたくさんありました。私たちは写真を撮って、お弁当を食べました。とても楽しい一日でした。

中文：

今天天气很好。早上有点冷，但是中午变暖了。我和朋友去了公园。公园里有很多花。我们拍了照片，吃了便当。是非常开心的一天。

### 文章 7：餐厅

昨日、友達と駅の近くのレストランへ行きました。その店は有名です。私はカレーを食べました。友達は魚料理を食べました。カレーは少し辛かったですが、とてもおいしかったです。店の人も親切でした。また行きたいです。

中文：

昨天我和朋友去了车站附近的餐厅。那家店很有名。我吃了咖喱。朋友吃了鱼料理。咖喱有点辣，但是很好吃。店员也很亲切。我还想再去。

### 文章 8：旅行

来月、友達と京都へ行きます。新幹線で行きます。京都でお寺を見たり、おいしい料理を食べたりします。写真もたくさん撮りたいです。旅行の前に、ホテルを予約しなければなりません。とても楽しみです。

中文：

下个月我和朋友去京都。坐新干线去。在京都参观寺庙、吃好吃的料理。也想拍很多照片。旅行前必须预订酒店。非常期待。

### 文章 9：学习日语

私は毎日日本語を勉強しています。朝、単語を覚えます。昼休みに日本語の音声を聞きます。夜、文法を勉強して、短い文章を書きます。日本語は難しいですが、毎日少しずつ勉強したら、上手になると思います。

### 文章 10：看病

今朝、頭が痛かったです。熱も少しありました。会社へ行かないで、病院へ行きました。医者は「今日はゆっくり休んでください」と言いました。薬をもらって、家へ帰りました。明日は元気になりたいです。

中文：

今天早上头疼。也有点发烧。我没有去公司，而是去了医院。医生说：“今天请好好休息。”我拿了药，回家了。希望明天能恢复精神。

私の学校は駅の近くにあります。教室は三階です。毎日九時から十二時まで日本語を勉強します。先生はとても親切です。授業の後で、友達と図書館へ行きます。そこで宿題をしたり、本を読んだりします。

今日は私の誕生日です。友達がケーキを買ってくれました。夜、家で小さいパーティーをしました。みんなで歌を歌って、写真を撮りました。プレゼントももらいました。とてもうれしかったです。

### 文章 14：电话

もしもし、田中さんですか。はい、田中です。明日、一緒に図書館へ行きませんか。すみません、明日は用事があります。では、日曜日はどうですか。日曜日は大丈夫です。じゃ、十時に駅で会いましょう。

### 文章 15：我的房间

私の部屋は二階にあります。部屋には机と椅子とベッドがあります。机の上にパソコンと日本語の本があります。窓の近くに小さいテレビがあります。部屋は広くないですが、とても好きです。

中文：

## 7. 背诵顺序建议

1. 先背判断句、助词、动词ます形、て形、ない形、た形。
2. 再背时间、地点、交通、食物、日常动词。
3. 每天背 1 篇短文，背完后默写。
4. 每篇文章必须做到：能读、能翻译、能听懂、能替换关键词造句。
5. 单词不要孤立背，要放进例句里背。
