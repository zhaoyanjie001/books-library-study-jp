## 2. 数字的组合 (Sūji no Kumiawase)

### 2.1. 两位数 (Jū no Kurai)

*   **11-19**: 十 + 个位数 (例: 11 = じゅういち)
*   **20-90**: 两位数 + 十 (例: 20 = にじゅう, 30 = さんじゅう)
*   **特殊**:
    *   40: よんじゅう
    *   70: ななじゅう
    *   90: きゅうじゅう

| 数字 | 日语读法 (平假名) | 日语读法 (罗马字) |
|---|---|---|
| 11 | じゅういち | juuichi |
| 12 | じゅうに | juuni |
| 13 | じゅうさん | juusan |
| 14 | じゅうよん | juuyon |
| 15 | じゅうご | juugo |
| 20 | にじゅう | nijuu |
| 25 | にじゅうご | nijuugo |
| 30 | さんじゅう | sanjuu |
| 40 | よんじゅう | yonjuu |
| 50 | ごじゅう | gojuu |
| 60 | ろくじゅう | rokujuu |
| 70 | ななじゅう | nanajuu |
| 80 | はちじゅう | hachijuu |
| 90 | きゅうじゅう | kyuujuu |
| 99 | きゅうじゅうきゅう | kyuujuukyuu |

### 2.2. 三位数 (Hyaku no Kurai)

*   **100**: ひゃく
*   **200-900**: 数字 + ひゃく
*   **特殊音变**:
    *   300: さんびゃく (sanbyaku)
    *   600: ろっぴゃく (roppyaku)
    *   800: はっぴゃく (happyaku)

| 数字 | 日语读法 (平假名) | 日语读法 (罗马字) |
|---|---|---|
| 100 | ひゃく | hyaku |
| 150 | ひゃくごじゅう | hyakugojuu |
| 200 | にひゃく | nihyaku |
| 300 | さんびゃく | sanbyaku |
| 400 | よんひゃく | yonhyaku |
| 500 | ごひゃく | gohyaku |
| 600 | ろっぴゃく | roppyaku |
| 700 | ななひゃく | nanahyaku |
| 800 | はっぴゃく | happyaku |
| 900 | きゅうひゃく | kyuuhyaku |

### 2.3. 四位数 (Sen no Kurai)

*   **1,000**: せん
*   **2,000-9,000**: 数字 + せん
*   **特殊音变**:
    *   3,000: さんぜん (sanzen)
    *   8,000: はっせん (hassen)

| 数字 | 日语读法 (平假名) | 日语读法 (罗马字) |
|---|---|---|
| 1,000 | せん | sen |
| 1,200 | せんにひゃく | sennihyaku |
| 2,000 | にせん | nisen |
| 3,000 | さんぜん | sanzen |
| 4,000 | よんせん | yonsen |
| 5,000 | ごせん | gosen |
| 6,000 | ろくせん | rokusen |
| 7,000 | ななせん | nanasen |
| 8,000 | はっせん | hassen |
| 9,000 | きゅうせん | kyuusen |

### 2.4. 万位数 (Man no Kurai)

*   日语的计数单位是“万”，而不是“千”。
*   10,000 = いちまん
*   100,000 = じゅうまん
*   1,000,000 = ひゃくまん
*   10,000,000 = せんまん

| 数字 | 日语读法 (平假名) | 日语读法 (罗马字) |
|---|---|---|
| 10,000 | いちまん | ichiman |
| 20,000 | にまん | niman |
| 100,000 | じゅうまん | juuman |
| 500,000 | ごじゅうまん | gojuuman |
| 1,000,000 | ひゃくまん | hyakuman |
| 10,000,000 | せんまん | senman |
| 100,000,000 | いちおく | ichioku |

## 3. 量词 (Josūshi / Counters)

量词在日语中非常重要，不同的物品、人、时间等都有对应的量词。

### 3.1. 通用量词 (General Counters)

|---|---|---|---|---|
| 1个 | 一つ | ひとつ | hitotsu | 用于不规则计数或小件物品 |
| 2个 | 二つ | ふたつ | futatsu | |
| 3个 | 三つ | みっつ | mittsu | |
| 4个 | 四つ | よっつ | yottsu | |

| 6个 | 六つ | むっつ | muttsu | |

### 3.3. 物品 (Items)

*   **～個 (～こ)**: 用于小而圆的物品 (苹果、鸡蛋、球等)
    *   1個: いっこ (ikko)
    *   2個: にこ (niko)
    *   3個: さんこ (sanko)
    *   何個: なんこ (nanko)
*   **～本 (～ほん/ぼん/ぽん)**: 用于细长物品 (铅笔、瓶子、树、伞等)
    *   1本: いっぽん (ippon)
    *   2本: にほん (nihon)
    *   3本: さんぼん (sanbon)
    *   何本: なんぼん (nanbon)
*   **～枚 (～まい)**: 用于薄而平的物品 (纸、盘子、衣服、邮票等)
    *   1枚: いちまい (ichimai)
    *   何枚: なんまい (nanmai)
*   **～冊 (～さつ)**: 用于书籍、杂志、笔记本等
    *   1冊: いっさつ (issatsu)
    *   何冊: なんさつ (nansatsu)
*   **～台 (～だい)**: 用于机器、车辆 (汽车、电视、电脑等)
    *   1台: いちだい (ichidai)
    *   何台: なんだい (nandai)
*   **～杯 (～はい/ばい/ぱい)**: 用于杯、碗、碗装的食物 (咖啡、汤等)
    *   1杯: いっぱい (ippai)
    *   2杯: にはい (nihai)
    *   3杯: さんばい (sanbai)
    *   何杯: なんばい (nanbai)
*   **～匹 (～ひき/びき/ぴき)**: 用于小动物 (猫、狗、鱼、虫等)
    *   1匹: いっぴき (ippiki)
    *   2匹: にひき (nihiki)
    *   3匹: さんびき (sanbiki)
    *   何匹: なんびき (nanbiki)
*   **～羽 (～わ)**: 用于鸟类、兔子
    *   1羽: いちわ (ichiwa)
    *   何羽: なんわ (nanwa)

### 3.4. 时间 (Time)

*   **～時 (～じ)**: 小时
    *   1時: いちじ (ichiji)
    *   4時: よじ (yoji)
    *   7時: しちじ (shichiji)
    *   9時: くじ (kuji)
    *   何時: なんじ (nanji)
*   **～分 (～ふん/ぷん)**: 分钟
    *   1分: いっぷん (ippun)
    *   2分: にふん (nifun)
    *   3分: さんぷん (sanpun)
    *   4分: よんぷん (yonpun)
    *   5分: ごふん (gofun)
    *   6分: ろっぷん (roppun)
    *   7分: ななふん (nanafun)
    *   8分: はっぷん (happun)
    *   9分: きゅうふん (kyuufun)
    *   10分: じゅっぷん / じっぷん (juppun / jippun)
    *   何分: なんぷん (nanpun)
*   **～半 (～はん)**: 半小时
    *   1時半: いちじはん (ichijihan)
*   **～秒 (～びょう)**: 秒
    *   1秒: いちびょう (ichibyou)
*   **～日 (～にち/か)**: 天数/日期
    *   1日: ついたち (tsuitachi) - 1号
    *   2日: ふつか (futsuka) - 2号 / 2天
    *   3日: みっか (mikka) - 3号 / 3天
    *   4日: よっか (yokka) - 4号 / 4天
    *   5日: いつか (itsuka) - 5号 / 5天
    *   6日: むいか (muika) - 6号 / 6天
    *   7日: なのか (nanoka) - 7号 / 7天
    *   8日: ようか (youka) - 8号 / 8天
    *   9日: ここのか (kokonoka) - 9号 / 9天
    *   10日: とおか (tooka) - 10号 / 10天
    *   14日: じゅうよっか (juuyokka) - 14号 / 14天
    *   20日: はつか (hatsuka) - 20号 / 20天
    *   24日: にじゅうよっか (nijuuyokka) - 24号 / 24天
    *   何日: なんにち (nannichi) - 几号 / 几天
*   **～週間 (～しゅうかん)**: 周数
    *   1週間: いっしゅうかん (isshuukan)
    *   何週間: なんしゅうかん (nanshuukan)
*   **～ヶ月 (～かげつ)**: 月数
    *   1ヶ月: いっかげつ (ikkagetsu)
    *   何ヶ月: なんかげつ (nankagetsu)
*   **～年 (～ねん)**: 年数
    *   1年: いちねん (ichinen)
    *   何年: なんねん (nannen)
*   **～回 (～かい)**: 次数
    *   1回: いっかい (ikkai)
    *   何回: なんかい (nankai)

### 3.5. 楼层 (Floor)

*   **～階 (～かい/がい)**: 楼层
    *   1階: いっかい (ikkai)
    *   2階: にかい (nikai)
    *   3階: さんがい (sangai)
    *   何階: なんがい (nangai)

### 3.6. 岁数 (Age)

*   **～歳 (～さい)**: 岁
    *   1歳: いっさい (issai)
    *   8歳: はっさい (hassai)
    *   10歳: じゅっさい / じっさい (jussai / jissai)
    *   20歳: はたち (hatachi) - 特殊读法
    *   何歳: なんさい (nansai)
    *   おいくつ: 几岁 (更礼貌)

### 3.7. 顺序 (Order)

*   **～番 (～ばん)**: 第～
    *   1番: いちばん (ichiban)
    *   何番: なんばん (nanban)

### 3.8. 其他常用量词

*   **～着 (～ちゃく)**: 用于衣服 (套)
*   **～足 (～そく)**: 用于鞋子、袜子 (双)
*   **～軒 (～けん)**: 用于房屋
*   **～件 (～けん)**: 用于事件、项目
*   **～通 (～つう)**: 用于信件
*   **～課 (～か)**: 用于课 (章节)
*   **～語 (～ご)**: 用于语言
*   **～円 (～えん)**: 日元
    *   100円: ひゃくえん (hyakuen)
    *   何円: なんえん (nan'en)
