# N5 助词 (Particles)

## 1. 格助词 (Case Particles)

### 1.1. が (ga)

*   **使用场景**:
    *   表示主语 (当主语不是话题时，或强调主语时)。
        *   例: わたし**が**いきます。(我去。)
    *   表示愿望、能力、喜好等动词或形容词的对象。
        *   例: にほんご**が**わかります。(我懂日语。)
        *   例: ラーメン**が**たべたいです。(我想吃拉面。)
        *   例: えいが**が**すきです。(我喜欢电影。)
    *   表示存在句的主语 (有生命或无生命)。
        *   例: ねこ**が**います。(有猫。)
        *   例: ほん**が**あります。(有书。)

### 1.2. を (o)

*   **使用场景**:
    *   表示他动词的直接宾语。
        *   例: パン**を**たべます。(吃面包。)
        *   例: コーヒー**を**のみます。(喝咖啡。)
    *   表示移动动词的移动范围或经过的场所。
        *   例: みち**を**あるきます。(在路上走。)
        *   例: こうえん**を**さんぽします。(在公园散步。)

### 1.3. に (ni)

*   **使用场景**:
    *   表示时间点 (具体时间)。
        *   例: ７時**に**おきます。(7点起床。)
    *   表示地点 (存在、到达、进入、放置)。
        *   例: へや**に**います。(在房间里。)
        *   例: がっこう**に**いきます。(去学校。)
        *   例: つくえ**に**ほんをおきます。(把书放在桌子上。)
    *   表示动作的对象或方向。
        *   例: ともだち**に**あいます。(和朋友见面。)
        *   例: せんせい**に**ききます。(问老师。)
    *   表示动作的结果或变化。
        *   例: いしゃ**に**なります。(成为医生。)
    *   表示授予或接受的对象。
        *   例: わたしはせんせい**に**プレゼントをあげます。(我给老师礼物。)
        *   例: わたしはともだち**に**ほんをもらいました。(我从朋友那里得到了书。)

### 1.4. へ (e)

*   **使用场景**:
    *   表示方向或目的地 (与「に」相似，但「へ」更强调方向性)。
        *   例: とうきょう**へ**いきます。(去东京。)
        *   例: うち**へ**かえります。(回家。)

### 1.5. で (de)

*   **使用场景**:
    *   表示动作发生的场所。
        *   例: レストラン**で**ごはんをたべます。(在餐厅吃饭。)
        *   例: こうえん**で**さんぽします。(在公园散步。)
    *   表示工具、手段或材料。
        *   例: はし**で**たべます。(用筷子吃。)
        *   例: バス**で**がっこうへいきます。(坐巴士去学校。)
        *   例: にほんご**で**はなします。(用日语说。)
    *   表示原因。
        *   例: かぜ**で**がっこうをやすみました。(因为感冒请假没去学校。)
    *   表示范围或总数中的极限。
        *   例: クラス**で**いちばんせがたかいです。(班级里最高。)

### 1.6. と (to)

*   **使用场景**:
    *   表示并列 (和...和...)。
        *   例: わたし**と**ともだち。(我和朋友。)
    *   表示共同行动的对象 (和...一起)。
        *   例: ともだち**と**えいがをみます。(和朋友看电影。)
    *   表示引用内容。
        *   例: 「おはよう」**と**いいました。(说了“早上好”。)
    *   表示比较的对象。
        *   例: にほんご**と**えいご**と**どちらがおもしろいですか。(日语和英语哪个更有趣？)

### 1.7. から (kara)

*   **使用场景**:
    *   表示起点 (时间、地点)。
        *   例: ９時**から**べんきょうします。(从9点开始学习。)
        *   例: とうきょう**から**きました。(从东京来的。)
    *   表示原因、理由。
        *   例: あめがふる**から**、いきません。(因为下雨，所以不去。)

### 1.8. まで (made)

*   **使用场景**:
    *   表示终点 (时间、地点)。
        *   例: ５時**まで**はたらきます。(工作到5点。)
        *   例: とうきょう**まで**いきます。(到东京。)

## 2. 提示助词 (Adverbial Particles / Topic Particles)

### 2.1. は (wa)

*   **使用场景**:
    *   提示话题或主语 (强调话题，而不是主语本身)。
        *   例: わたし**は**がくせいです。(我是学生。)
        *   例: このほん**は**おもしろいです。(这本书很有趣。)
    *   表示对比或强调。
        *   例: コーヒー**は**のみますが、おちゃ**は**のみません。(咖啡我喝，但茶不喝。)

### 2.2. も (mo)

*   **使用场景**:
    *   表示“也”、“都”。
        *   例: わたし**も**がくせいです。(我也是学生。)
        *   例: みんな**も**いきます。(大家也都去。)
    *   表示强调数量之多或之少。
        *   例: １００えん**も**あります。(竟然有100日元。)
        *   例: １えん**も**ありません。(连1日元也没有。)

## 3. 终助词 (Sentence-ending Particles)

### 3.1. か (ka)

*   **使用场景**:
    *   表示疑问。
        *   例: これはなんですか。(这是什么？)
        *   例: いきます**か**。(去吗？)

### 3.2. ね (ne)

*   **使用场景**:
    *   表示征求对方同意或确认，带有亲切感。
        *   例: いいおてんきです**ね**。(天气真好啊。)
        *   例: そうです**ね**。(是啊。)

### 3.3. よ (yo)

*   **使用场景**:
    *   表示告知对方不知道的信息，带有强调或提醒的语气。
        *   例: あしたはやすみです**よ**。(明天是休息日哦。)
        *   例: おいしいです**よ**。(很好吃哦。)

## 4. 接续助词 (Conjunctive Particles)

### 4.1. て (te)

*   **使用场景**:
    *   连接动词或形容词，表示动作的顺序、并列、原因、手段等。
        *   例: ごはんをたべ**て**、ほんをよみます。(吃了饭，然后读书。)
        *   例: きれい**で**、しんせつなひと。(漂亮又亲切的人。)
        *   例: あめがふっ**て**います。(正在下雨。)

### 4.2. が (ga)

*   **使用场景**:
    *   表示转折 (但是)。
        *   例: にほんごはむずかしいです**が**、おもしろいです。(日语很难，但是很有趣。)

### 4.3. から (kara)

*   **使用场景**:
    *   表示原因、理由 (因为...所以...)。
        *   例: あめがふる**から**、いきません。(因为下雨，所以不去。)

### 4.4. し (shi)

*   **使用场景**:
    *   表示并列原因或理由 (而且，又)。
        *   例: やすい**し**、おいしいです。(又便宜又好吃。)

## 5. 其他助词

### 5.1. の (no)

*   **使用场景**:
    *   表示所属、修饰关系。
        *   例: わたし**の**ほん。(我的书。)
        *   例: にほんご**の**せんせい。(日语老师。)
    *   将动词或形容词短语名词化。
        *   例: ほんをよむ**の**がすきです。(喜欢读书。)

### 5.2. や (ya)

*   **使用场景**:
    *   表示不完全列举 (等等)。
        *   例: ほん**や**ペンなどがあります。(有书、笔等等。)

### 5.3. など (nado)

*   **使用场景**:
    *   表示等等 (常与「や」一起使用)。
        *   例: ほんやペン**など**があります。(有书、笔等等。)

### 5.4. だけ (dake)

*   **使用场景**:
    *   表示限定 (只，仅仅)。
        *   例: わたし**だけ**いきます。(只有我去。)

### 5.5. しか (shika)

*   **使用场景**:
    *   表示限定 (只，仅仅)，后接否定。
        *   例: １００えん**しか**ありません。(只有100日元。)

### 5.6. ぐらい / くらい (gurai / kurai)

*   **使用场景**:
    *   表示大约、左右。
        *   例: １時間**ぐらい**かかります。(大约需要1小时。)

### 5.7. ほど (hodo)

*   **使用场景**:
    *   表示程度、大约。
        *   例: １００えん**ほど**です。(大约100日元。)
    *   表示“没有...那么...” (后接否定)。
        *   例: にほんごはむずかしいですが、おもった**ほど**むずかしくありません。(日语虽然难，但没有想象中那么难。)

### 5.8. など (nado)

*   **使用场景**:
    *   表示等等 (常与「や」一起使用)。
        *   例: ほんやペン**など**があります。(有书、笔等等。)
