# ところ

## 1. 所有接续

### 名词用法

- `ところ`
- `この / その / あの / どの + ところ`
- `名词 + の + ところ`
- `动词普通形 + ところ`
- `い形容词普通形 + ところ`
- `な形容词な + ところ`
- `名词の + ところ`

例：

- 私のところに来てください。
- そのところはもう一度説明します。

### 表示动作阶段、时间点

- `动词辞书形 + ところ`：正要做某事
- `动词ている形 + ところ`：正在做某事
- `动词た形 + ところ`：刚做完某事
- `动词辞书形 + ところだった`：差一点就要做某事
- `动词た形 + ところだった`：差一点发生某事，常用于险些出现的结果

例：

- 今から出かけるところです。
- 今、資料を読んでいるところです。
- ちょうど仕事が終わったところです。
- もう少しで電車に乗り遅れるところだった。

### 表示情况、范围、程度

- `普通形 + ところ`
- `动词た形 + ところ`
- `形容词 / 名词修饰形 + ところ`
- `动词た形 + ところ`：一……结果……
- `动词た形 + ところで`：即使……也……
- `动词辞书形 + ところまで`：到……的程度
- `动词普通形 + ところを見ると`：从……来看
- `名词 + の + ところでは`：据……所知
- `名词 + の + ところによると`：根据……的说法

例：

- 調べたところ、問題はありませんでした。
- 私が知っているところでは、明日は休みです。
- 彼の話すところによると、試験は難しかったそうです。

### 和助词结合的常见形式

- `ところに`：正在某个时点时，后项发生。
- `ところへ`：正在某个时点时，有人来或某事发生。
- `ところを`：在某个场面下；明知对方处于某种状态仍打扰或感谢。
- `ところで`：即使……也……；话说回来。
- `ところが`：可是、然而。
- `どころか`：岂止……，反而……。
- `どころではない`：不是做……的时候；顾不上……。

例：

- 出かけようとしたところに、電話がかかってきた。
- お忙しいところを、ありがとうございます。
- 今は遊んでいるどころではない。

## 2. 意思

`ところ` 的核心意思是“地方、位置、部分”，也可以从“某个点”引申为“时间点、阶段、情况、范围”。

常见意思：

- 地方、场所：地方、位置。
- 部分、点：内容中的某一点。
- 时间点、阶段：正要、正在、刚刚。
- 情况、状态：某种场面或局面。
- 范围、限度：就某人所知、就某个范围而言。
- 结果、发现：做了某事后得到某个结果。
- 逆接、转折：可是、即使。
- 程度：到某种程度或地步。

## 3. 区别

### ところ vs 場所

`場所` 更偏向具体地点，常用于客观的“场所、位置”。`ところ` 更宽泛，可以表示地点，也可以表示抽象的“部分、阶段、情况”。

- 会議の場所を教えてください。
- この文のわからないところを教えてください。

### ところ vs ばかり

`たところ` 强调动作刚刚完成的时间点，常用于客观叙述。`たばかり` 强调说话人感觉“刚刚做完”，这个“刚刚”可以是主观感受，不一定真的很短。

- 今、家に着いたところです。
- 先月日本語を勉強し始めたばかりです。

### ところ vs うち

`ところ` 关注某个时间点或阶段。`うち` 强调在某个状态持续期间完成某事，常译为“趁着……”。

- 出かけるところです。
- 明るいうちに帰りましょう。

### ところ vs 最中

`ているところ` 和 `最中` 都可以表示“正在……”。`最中` 更强调动作进行的过程中，后面常接突发事件；`ところ` 更中性，可以单纯说明当前阶段。

- 今、会議をしているところです。
- 会議の最中に電話が鳴りました。

### ところ vs ところで

`ところ` 是名词或形式名词。`ところで` 可以表示转换话题，意思是“话说回来、对了”。

- 今、宿題をしているところです。
- ところで、明日の予定は何ですか。

### ところ vs どころか

`ところ` 表示地点、阶段、情况等。`どころか` 是固定表达，表示“岂止……，反而……”，后项通常比前项更进一步或相反。

- 今、説明しているところです。
- 休むどころか、昼ご飯も食べられませんでした。

## 4. 使用场景

### 表示地点

- 駅の近くのところに住んでいます。
- 荷物は机の上のところに置いてあります。

### 表示文章、话题、内容中的部分

- このところがよくわかりません。
- 大事なところに線を引いてください。

### 表示动作阶段

- ご飯を食べるところです。
- ご飯を食べているところです。
- ご飯を食べたところです。

### 表示调查、确认后的结果

- 先生に聞いたところ、明日の授業は休みだそうです。
- 確認したところ、予約は入っていませんでした。

### 表示差点发生

- もう少しで事故になるところだった。
- 危うく財布を忘れるところだった。

### 表示让步、逆接

- 今さら謝ったところで、もう遅い。
- いくら説明したところで、彼は納得しないだろう。

### 表示“正好、刚好”的场面

- いいところに来ましたね。
- 困っていたところ、友達が助けてくれました。

### 表示范围

- 私が知っているところでは、彼はまだ来ていません。
- 今のところ、問題はありません。

## 5. 常用表达

- `今のところ`：目前、现在来看。
- `いいところ`：好地方；好处；正好的时候。
- `悪いところ`：坏处、缺点。
- `知っているところでは`：据我所知。
- `聞いたところでは`：听说、据听到的情况。
- `ちょうど...ところ`：正好处于某个阶段。
- `...ところだった`：差一点就……。
- `...たところ`：一……结果……。
- `...たところで`：即使……也……。
- `...どころではない`：顾不上……。
- `...どころか`：岂止……，反而……。
